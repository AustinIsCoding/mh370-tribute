# MH370 — We Remember | Kami Mengenang | 我们铭记

A personal tribute website for the 239 souls of Malaysia Airlines Flight MH370, which disappeared on 8 March 2014.

Built with love by a Malaysian who cares.

## About

This is a non-commercial, personal memorial project. It is not affiliated with Malaysia Airlines or any government body.

**Live:** [mh370-tribute.vercel.app](https://mh370-tribute.vercel.app)

## Features

- **Trilingual** — English / Bahasa Malaysia / 中文 (toggle instantly)
- **239 animated stars** — one for every soul aboard
- **6 sections** — Hero, Timeline, The 239 Souls, Theories, Latest Updates, Tribute Wall
- **Tribute Wall** — leave a candle message for the 239
- **Fully static** — no backend, no database, deployable anywhere

## Stack

- [React](https://react.dev) + [Vite](https://vite.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- Fonts: Playfair Display · Lora · Nunito
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
npm install

```

## Build

```bash

```

## Deploy

